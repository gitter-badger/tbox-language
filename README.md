## Tbox - 你好T型箱子
```javascript
import console;
func main() {
	printf("Hello, Tbox");
}
```
打印输出：

```
Hello, Tbox
```

## Tbox - 算法
```javascript
import console;
func main() {
	printf(1+1);
}
```
打印输出：

```
2
```
---
## Tbox - 判断
```javascript
import console;
func main() {
    var x = 6;
	if(x > 5) {
		printf("Well done!");
	}
}
```
打印输出：
```
Well done!
```

## Tbox - 文本游戏
```javascript
import console;
func main() {
	string version = "0.0.1";
	string title = "Text Game";
	string text = "那天，这你来玩呢";
	printf(text);
	printf("你好，好耶，稍等");
	printf("稍等...");
}
```
打印输出：

```
那天，这你来玩呢
你好，好耶，稍等
稍等...
```
