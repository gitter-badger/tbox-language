## Tbox 你好T型箱子
```javascript
//import console;
import "console.tbox";
func main() {
	print("Hello, Tbox");
}
```
打印输出：

```

```

## Tbox 判断
```javascript
//import console;
import "console.tbox";
func main() {
    math x = 6;
	if(x > 5) {
		print("Well done!");
	}
}
```
打印输出：
```
Well done!
```

## Tbox 文本游戏
```javascript
//import console;
import "console.tbox";
func main() {
	string version = "0.0.1";
	string title = "Text Game";
	string text = "那天，这你来玩呢";
	print(text);
	print("你好，好耶，稍等");
	print("稍等...");
}
```
打印输出：

```
那天，这你来玩呢
你好，好耶，稍等
稍等...
```
