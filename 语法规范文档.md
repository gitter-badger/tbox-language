# Tbox 语法规范

---
## Tbox - 你好T型箱子
```javascript
//import console;
import "console.tbox";
func main() {
	print("Hello, Tbox");
}
```
打印输出：

```
Hello, Tbox
```
----
## Tbox - 算法
```javascript
//import console;
import "console.tbox";
func main() {
	math i = 1 + 1;
	print(i);
	print(50/20);
}
```
打印输出：

```
2
2.5
```
----
## Tbox - 你好算法
```javascript
//import console;
import "console.tbox";
func main() {
	math i = 50 / 20;
	string he = "hello, " + i;
	print(he);
}
```
打印输出：

```
hello, 2.5
```
